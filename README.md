# Sneha Bangargi - Creative Portfolio Website

This package contains a complete one-page portfolio website for IT Business Analysis, Project Management, Product Owner support, PMO, Delivery and Consulting roles.

## Files

- `index.html` - main website
- `styles.css` - full styling
- `assets/` - SVG visuals and logo
- `assets/cv/` - role-specific CV PDFs included as downloadable files

## How to publish

### Option 1: GitHub Pages
1. Create a new GitHub repository.
2. Upload all files and folders from this package.
3. Go to Settings > Pages.
4. Select the main branch and root folder.
5. Save and wait for the website link.

### Option 2: Netlify
1. Go to Netlify.
2. Drag and drop this full folder.
3. Netlify will publish it automatically.

## Important privacy note
