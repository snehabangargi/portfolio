# Sneha Bangargi - Creative Portfolio Website, One CV Version

This package contains a complete one-page portfolio website for IT Business Analysis, Project Management, PMO, Delivery and Consulting roles.

## Files

- `index.html` - main website
- `styles.css` - full styling
- `assets/` - SVG visuals and logo
- `assets/cv/Sneha_Bangargi_IT_Business_Analyst_CV.pdf` - one focused CV download

## CV positioning

The website now uses only one CV: an IT Business Analyst-led CV that can also support Project Coordinator, PMO Analyst, Product Owner support, Consulting Analyst and Digital Transformation applications.

## Publishing

Upload this whole folder to GitHub Pages, Netlify or Vercel.

## Privacy note

The website includes the contact details already present in the CV. Remove the phone number from the footer if you want the public version to be more private.
